Start here,
end anywhere.
Upon launching the H-D® Ride Planner on their desktop, a user would be able to create and customize a ride by right-clicking points on a map that they wished to stop.
Using imported routes from the H-D® Owners Group booklet, anyone can find suggestions from H-D® on some of the best roads to ride in the USA. After creating and saving rides using the desktop application, the user can then follow along and check-in using their mobile device.
